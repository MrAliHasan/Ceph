WCeph v1 File Format Specification
===================================

WCeph Version 1 File Format |    |           
---------------|------------------
Version        | 1
File Extension | `.wceph`
File Format    | ZIP
ZIP Encoding   | UTF-8

## ZIP file structure
```
root
|__ index.json
|__ images
    |__ image1.png
    |__ image2.jpg
    |__ any_name.jpg  
```
